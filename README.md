# Hundo: A Coding Project

**Live life one loop at a time.**  

A small JavaScript exercise that displays numbers from 0 to 100, with **even numbers in bold**.

## Features
- HTML structure with a clean layout
- CSS and Bootstrap styling
- JavaScript fundamentals:
  - Loops
  - Functions
  - DOM manipulation
  - If/Then/Else statements
  - Boolean logic

## How to Use
Open `index.html` in your browser and see the numbers displayed dynamically.
